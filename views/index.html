üîç What GEDquality Checks

GEDquality performs comprehensive integrity checks including:

  • Family Membership: Ensures each person is in at least one family (as parent or child)
  • Page Consistency: Verifies all people are from the same OFB page
  • Date Validation: Checks that dates are valid GEDCOM format
  • Date Logic: Detects inconsistencies like:
    • Birth after death
    • Children born before parents' marriage
    • Children born after mother's death
    • Excessive lifespans (>120 years)
    • Mother over 50 at child's birth
  • Given Names: Identifies unusual given names for German-speaking regions
  • Gender Consistency: Checks if gender matches the given name
  • Source References: Ensures each person has entry label source
  • Surname Consistency: Checks children have consistent surnames with father and siblings

üí° Tips

  • Fix critical issues first (date inconsistencies, missing family links)
  • Unusual given names may be valid variants or errors - review manually
  • Gender mismatches might indicate transcription errors
  • Surname inconsistencies often indicate adoptions or remarriages
